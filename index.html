01 — Project

Engineering Report

The Problem: Standard home watering systems operate on arbitrary timers — running for a fixed duration regardless of actual soil conditions. This wastes gallons of water daily through over-irrigation, especially in drought-prone climates.

01 — CAD Design Phase
⬡
Fusion 360 Render
Custom 3D-Printed Enclosure
Designed in Autodesk Fusion 360 (Educational License). Key feature: integrated mounting brackets for the Elegoo capacitive soil sensor — eliminating post-assembly alignment errors.
02 — Electronics
⬡
Breadboard Photo
Arduino C++ Control Logic
Elegoo Uno R3 reads analog soil moisture values. When threshold is crossed, the motor relay triggers the valve. A 5-line snippet of the core logic is shown below.
// Core irrigation trigger — reads sensor, activates pump int moisture = analogRead(A0); int threshold = 450; // ~15% soil moisture if (moisture > threshold) { digitalWrite(RELAY_PIN, HIGH); } else { digitalWrite(RELAY_PIN, LOW); }


03 — Data & Iteration Log
Iteration 1

Initial prototype used a mechanical valve assembly. Under pressure testing, the gasket seal failed at the junction with the printed housing.

✗ Mechanical valve leaked
Fix

Redesigned the gasket seat geometry in Fusion 360 — added a 1.2mm lip channel and re-printed with 40% infill. Seal held under 2x rated pressure.

✓ Redesigned & re-printed
Iteration 2

Code successfully triggered the motor relay based on the 15% moisture threshold. Pump activated within 200ms of sensor crossing the analog threshold value.

✓ System functional end-to-end

02 — Inspiration

The Singapore Connection

Systems efficiency as a design philosophy

Living in Singapore exposed me to infrastructure that treats resource efficiency not as a constraint, but as a design objective. The national water reclamation strategy, the MRT's real-time load-balancing, and the NEWater program all share a common principle: every unit of input should be accounted for and maximized.


This project applies that philosophy at the residential scale — replacing arbitrary timer-based irrigation with closed-loop sensor feedback. The goal is the same: eliminate waste through precision.
